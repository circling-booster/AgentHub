# Guides

AgentHub 개발을 위한 실전 가이드입니다. 구현 방법, 레시피, 튜토리얼을 제공합니다.

---

## 하위 폴더

| 폴더 | 설명 |
|------|------|
| **playground/** | Playground 사용 가이드 (백엔드 API 수동 테스트, SSE 디버깅) |
| **extension/** | Chrome Extension 개발 가이드 (컴포넌트 추가, API 호출, 디버깅) |
| **implementation/** | 일반 구현 레시피 (Entity, Service, Port, Adapter 작성 체크리스트) |
| **standards/** | 외부 표준 통합 가이드 (MCP, A2A, ADK 구현 방법) |
| **troubleshooting/** | 문제 해결 가이드 (Import 오류, 연결 문제, 디버깅) |

---

## Guides vs Architecture 구분

| 구분 | Guides (HOW) | Architecture (WHAT) |
|------|--------------|---------------------|
| 목적 | 구현 방법, 레시피, 튜토리얼 | 구조, 설계, 개념 설명 |
| 질문 | "이것을 어떻게 만드는가?" | "이것은 무엇인가?" |
| 예시 | Extension 컴포넌트 추가 방법 | Extension 컴포넌트 구조도 |
