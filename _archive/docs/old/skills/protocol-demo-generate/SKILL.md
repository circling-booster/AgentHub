# 데모 시나리오 생성

기능: `$ARGUMENTS`

## 데모 구성요소

### 1. 시나리오 스크립트
- 사용자 관점 스토리
- 단계별 입력/출력
- 예상 결과

### 2. 샘플 데이터
- 테스트 입력 데이터
- Mock 응답 (필요시)

### 3. 실행 스크립트
```bash
#!/bin/bash
# demo.sh
echo "=== Feature Demo ==="
# 실행 명령어
```

## 데모 형식

```markdown
# [기능명] 데모

## 시나리오
사용자가 [상황]에서 [기능]을 사용하여 [결과]를 얻는다.

## 단계

### Step 1: [제목]
**입력:**
[사용자 입력]

**출력:**
[시스템 응답]

### Step 2: [제목]
...

## 핵심 포인트
- [강조점 1]
- [강조점 2]
```

## 출력

- 데모 스크립트 (Markdown)
- 실행 가능한 예제 코드
- 스크린샷 가이드 (해당시)
