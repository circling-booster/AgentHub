# 디버깅

대상: `$ARGUMENTS`

## 디버깅 프로세스

### 1. 에러 정보 수집
- 에러 메시지 전문
- 스택 트레이스
- 재현 조건

### 2. 원인 분석
- 에러 발생 위치 파악
- 관련 코드 검토
- 최근 변경사항 확인

### 3. 가설 수립 및 검증
- 가능한 원인 목록
- 각 가설 검증 방법
- 디버그 로깅 추가

### 4. 해결책 제시
- 근본 원인 설명
- 수정 코드 제안
- 재발 방지 방안

## 출력 형식

```markdown
## 에러 분석

### 증상
[에러 메시지]

### 원인
[근본 원인 설명]

### 해결책
[수정 방법]

### 수정 코드
[코드 변경]

### 예방
[재발 방지 방안]
```
